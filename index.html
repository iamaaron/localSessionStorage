Local & Session Storage Case Study
Summary

This SPA utilizes localStorage and sessionStorage via the HTML5 WebStorage API.

localStorage

The idea behind this case study is to be able to add items to localStorage and be able to view them through the table below. You may also be able to remove keys through the form below, as well as completely clear out localStorage.

sessionStorage

Another idea behind this case study is to show off sessionStorage as well, which is per the browsers session. Hence the tab session. Try clearing the sessionStorage via the button below and open this page in a new tab.

There is a table and form controls below to allow you to see localStorage in action. Or, the developer console also is an option to see what's going on when you update the form fields by entering localStorage into the console. Naturally, try opening and closing the browser tab and application without clearing your cache to see localStorage shine.
